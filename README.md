# Intelligent Traffic Congestion Prediction and Signal Timing Optimization

## 📌 Overview

This project predicts traffic congestion using Machine Learning and optimizes traffic signal timing using a Genetic Algorithm to reduce waiting time and improve traffic flow.

---

## 🎯 Objectives

* Predict traffic congestion using ML
* Optimize signal timing dynamically
* Reduce vehicle waiting time
* Improve traffic efficiency

---

## 🧠 Technologies Used

* Python
* Random Forest
* Genetic Algorithm
* Pandas, NumPy

---

## 📊 Results

* Fixed System: 240 sec
* Optimized System: 140 sec
